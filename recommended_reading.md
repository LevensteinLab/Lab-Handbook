# Recommended Reading

*Some recommended foundational papers and reviews to get acquainted with selected topics of interest to the lab*

**Sleep**

- Stickgold R, 2005, Nature. [Sleep-dependent memory consolidation](https://www.sciencedirect.com/science/article/pii/S0896627304005409)
- Diekelmann S & Born J, 2010, Nature Reviews Neuroscience. [The memory function of sleep](https://www.nature.com/articles/nrn2762)
- Lewis PA & Durrant SJ, 2011, Trends in Cognitive Sciences. [Overlapping memory replay during sleep builds cognitive schemata](https://www.sciencedirect.com/science/article/pii/S1364661311001094)
- Rasch B & Born J, 2013, Physiological Reviews. [About sleep’s role in memory](https://pubmed.ncbi.nlm.nih.gov/23589831/)
- Frank MG, 2014, Current Topics in Behavioral Neurosciences. [Sleep and synaptic plasticity in the developing and adult brain](https://link.springer.com/chapter/10.1007/7854_2014_305)
- Klinzing JG et al., 2019, Nature Neuroscience. [Mechanisms of systems memory consolidation during sleep](https://www.nature.com/articles/s41593-019-0467-3)
- Sun L et al., 2020, Philosophical Transactions. [Experience and sleep-dependent synaptic plasticity: from structure to activity](https://royalsocietypublishing.org/rstb/article/375/1799/20190234/23743/Experience-and-sleep-dependent-synaptic-plast)
- Brodt S et al., 2023, Neuron. [Sleep — a brain state serving systems memory consolidation](https://www.sciencedirect.com/science/article/pii/S0896627323002015)
- Lutz ND et al., 2026, Physiological Reviews. [Sleep’s contribution to memory formation](https://journals.physiology.org/doi/abs/10.1152/physrev.00054.2024)

**Memory consolidation**
